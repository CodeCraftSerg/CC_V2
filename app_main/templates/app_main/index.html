Main Features Of CloudyContacts

Our application offers a variety of features, with the main ones described below.

User authentication

The application has an authentication mechanism and provides access to basic functions only for registered users. Each registered user has access only to their own data, files, and other resources.

Learn More

Contacts

An authorized user can maintain their own contact book. For each contact, you can add all the necessary information, such as names, phone numbers, addresses, emails, birthdays, social network profiles, etc.

Learn More

News

An authorized user can view the latest news, weather forecasts, currency rates, etc.




Learn More

Files

Authorized users can upload their files to cloud storage and access them. The application allows users to sort files by category (images, documents, videos, etc.).
